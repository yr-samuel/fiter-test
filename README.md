# Desafio Fiter

Este é um desafio, que engloba consumir um endpoint que traz os colaboradores da Fiter, e mostrá-los em formato de Card, e como a Fiter está crescendo, isso deve ser responsivo, pois a quantidade de colaboradres vem aumentando cada vez mais.

## O que foi usado?

- ReactJS c/ Typescript
- MaterialUI

## Forma de executar o projeto

1. Clone o projeto, da seguinte forma -> git clone git@github.com:yr-samuel/fiter-test.git
2. Abra um terminal a partir do diretório qual foi clona, e rode o comando -> npm i
3. Após a instalações dos pacotes necessários para a aplicação, basta rodar o comando -> npm start
4. A partir de agora, em algum momento irá abrir o navegador com o projeto em execução, caso isso não aconteça, basta ver que a aplicação de fato está em execução e abrir o navegador no endereço -> localhost:3000
